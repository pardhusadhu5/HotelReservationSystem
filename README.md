# LuxeHaven Hotels & Resorts - Hotel Reservation System

A premium, modern web application for hotel reservations, built as part of the CodeAlpha internship.

## Technologies Used
- **Frontend Core**: HTML5, Vanilla JavaScript (ES6)
- **Styling**: Modern CSS3 (Dark Theme, CSS Grid/Flexbox, Glassmorphism, Animations)
- **Icons**: SVG icons embedded inline or via CSS
- **Data Persistence**: HTML5 `localStorage`

## Features
- **Modern Dark UI**: Premium user interface with HSL tailored color schemes, gold gradients, glassmorphism card designs, and micro-interactions.
- **Dynamic Search & Filters**: Search rooms by check-in/check-out dates, room class (Standard, Deluxe, Executive Suite, Penthouse), pricing range, and minimum guest capacity.
- **Interactive Booking Flow**: Book rooms through an overlay modal with auto-calculated total prices based on nights stayed, complete with field validations and a mock checkout screen.
- **My Reservations Panel**: Retrieve, view, and cancel bookings in real time. Cancelling a booking instantly marks the room as available again.
- **Mock Review System**: Read existing reviews or write your own ratings/comments for each room, with instant updates.
- **Data Persistence**: All bookings, availability status, and reviews are saved directly inside your browser's local storage.
